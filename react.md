## 其它

#### - [在设备上运行(Android)](http://wiki.jikexueyuan.com/project/react-native/RunningOnDeviceAndroid.html)

通过 Wi-Fi 来配置设备并且连接上你的开发者服务器
要做到这一点，你的电脑和你的手机必须在同一个 wifi 网络下。

打开震动菜单 (摇动设备)
前往 Dev Settings
前往 Debug server host for device
输入该设备的 IP 和 Reload JS
